What if i have an existing struts application built using ant or maven and i want to deploy that application to GAE, there is no tutorial to explain how to port the existing struts web application to make it deployable on GAE. This post will explain how to do it.

1.Install Google App Engine SDK

Get the Google App Engine SDK. Unpack the archive in a convenient location on your hard drive.

Note: Unpacking the archive creates a directory whose name is something like appengine-java-sdk-X.X.X, where X.X.X is the SDK version number. Throughout this documentation, this directory will be referred to as appengine-java-sdk/.
